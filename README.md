# Ingeniería Millar — Sitio Web

Sitio web corporativo de Ingeniería Millar. Consultoría técnica en ingeniería minera, Chile.

## Stack

- Astro 5 (salida estática)
- React 19 (islands interactivos)
- Tailwind CSS v4
- Deploy: Vercel

## Comandos

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # → dist/
npm run preview
```

## Contenido editable

Todo el texto, imágenes y datos del sitio están centralizados en `src/data/site.ts`.

## Deploy

Vercel detecta Astro automáticamente (build: `npm run build`, output: `dist/`).
Dominio: `ingenieriamillar.cl`.
